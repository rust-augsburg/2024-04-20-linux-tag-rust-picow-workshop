# Rust Embedded

Herausforderungen

* `no_std`-Umgebung
* Cross-Compilierung
* Debugging
* Logging

Möglichkeiten in Rust

 * `rp-pico`
   * einfach(er) zu Lernen
   * Sammlung von Crates
 * [`embassy`](https://embassy.dev/)
   * async-basiert
   * schwieriger zu Lernen
   * unterstützt Wi-Fi (und interne LED)
